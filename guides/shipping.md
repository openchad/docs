# Build as Standalone Program

```sh
npm run build
```

## Go to release folder
To distribute the application, bundle everything inside the Release/** directory. 
You can package it as a .zip, .tar.gz, or .rar archive, or wrap it in an installer.
```
Release/
├── Backend/
├── frontend/ 
├── Backend/
├── ModelProvider/ 
└── Pipeline/
└── python/
└── Settings/
└── Tools/
└── your-program
```
