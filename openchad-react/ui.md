## Standard UI Components

`shadcn/ui` The library re-exports a comprehensive suite of modular components. These are optimized for performance, accessibility, and high visual appeal:

| Component | Description |
| :--- | :--- |
| **`AspectRatio`** | Forces child elements to maintain a specific aspect ratio (e.g., `16:9`, `1:1`). |
| **`Avatar`** | User profile image container with fallback initials and styling. |
| **`Badge`** | Decorative status labels and inline tags. |
| **`Button`** | Standard interactive button with various variants (primary, secondary, outline, ghost). |
| **`Calendar`** | Full-featured calendar interface for dates and schedules. |
| **`Card`** | Container component with `CardHeader`, `CardTitle`, `CardDescription`, `CardContent`, and `CardFooter`. |
| **`Chart`** | Integrated charting elements powered by standard drawing libraries. |
| **`Checkbox`** | Multi-state checkbox with custom icons and animations. |
| **`Command`** | Command palette dialogue container for searching options and actions. |
| **`ContextMenu`** | Custom right-click overlay dropdown menus. |
| **`Dialog`** | Accessible overlay modals that appear on top of other content. |
| **`DropdownMenu`** | Floating menus triggered by buttons or click events. |
| **`Empty`** | Standard placeholder illustration and message for empty data states. |
| **`HoverCard`** | Reveals preview information when hovering over an element. |
| **`Input`** | Standard text and data input elements. |
| **`Label`** | Form labels styled and associated with form inputs. |
| **`Popover`** | Floating content boxes that appear next to a trigger element. |
| **`Progress`** | Visual progress bar supporting indeterminate states. |
| **`RadioGroup`** | Set of radio inputs where only one can be active at a time. |
| **`Resizable`** | Layout wrappers that can be resized by dragging split-bars. |
| **`ScrollArea`** | Custom scrollbars that look beautiful on all platforms (Windows, macOS). |
| **`Select`** | Dropdown option picker list. |
| **`Slider`** | Range input selectors. |
| **`Sonner`** | Advanced toast notification system. |
| **`Spinner`** | Circular visual loading indicator. |
| **`Switch`** | Toggle switch input (on/off toggles). |
| **`Table`** | Fully styled table grid elements (`TableHeader`, `TableBody`, `TableCell`, etc.). |
| **`Tooltip`** | Micro-help bubbles that appear on hover. |
